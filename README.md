# Blog-App
Simple blog application created using Ruby on Rails 

![image](./screen.png)


## Built With

- Ruby 2.7.1

- Ruby on Rails 5.2.3

## Getting Started

- Clone the repository

- Use the command "rails server" to run the project

- Enter in the localhost: localhost:3000

- Create, edit and delete post

### Prerequisites

- Ruby 2.7.1

- Ruby on Rails 5.2.3
